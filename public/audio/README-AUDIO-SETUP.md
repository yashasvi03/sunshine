## Audio Format Guidelines

- **Format:** MP3 (best browser compatibility)
- **Quality:** 128-192 kbps (balance quality and file size)
- **File size:** Keep under 5 MB per file for faster loading
- **Volume:** Normalize audio levels for consistency across pages

## How It Works

- Audio auto-plays when each page loads (with user interaction fallback)
- Loops continuously while on the page
- Stops automatically when navigating away
- Each page has its own unique background music to match the mood
