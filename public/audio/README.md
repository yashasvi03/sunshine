# Audio Files

## Required Files

### happy-birthday.mp3
Add a "Happy Birthday to You" song audio file here named `happy-birthday.mp3`.

The audio will:
- Auto-play when the birthday cake page loads
- Loop continuously in the background
- Enhance the birthday celebration experience

**Recommended format:** MP3
**Suggested duration:** 30-60 seconds (will loop)
**File size:** Keep under 5 MB for faster loading
