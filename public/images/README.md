# Timeline Photos

Place your 8 timeline photos here with these names:

- `moment-1.jpg` - First Meeting
- `moment-2.jpg` - First Date
- `moment-3.jpg` - First Trip
- `moment-4.jpg` - Special Celebration
- `moment-5.jpg` - Unforgettable Memory
- `moment-6.jpg` - Inside Joke Origin
- `moment-7.jpg` - Recent Adventure
- `moment-8.jpg` - Latest Memory

## Specifications

- **Format**: JPG or PNG
- **Resolution**: Minimum 1200x800px
- **Aspect Ratio**: 3:2 or 4:3 preferred (landscape)
- **File Size**: <500KB each (use compression tools if needed)
- **Orientation**: Landscape preferred

## Photo Optimization Tools

- [TinyPNG](https://tinypng.com/) - Free online compression
- [Squoosh](https://squoosh.app/) - Google's image optimizer
- Photoshop/GIMP - "Save for Web"

## Tips

- Use high quality photos with good lighting
- Make sure faces are clearly visible
- Choose photos that tell your story
- Landscape orientation works best for the polaroid layout
