# Birthday Website - 26th Celebration

An interactive, romantic birthday website built with React, Framer Motion, and modern web technologies.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Run development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

## Project Structure

- **Page 1**: The Unlock (password entry)
- **Page 2**: Birthday Facts (calendar flip)
- **Page 3**: Countdown (days/hours/minutes together)
- **Page 4**: 26 Reasons (balloon pop)
- **Page 5**: Timeline (photo memories)
- **Page 6**: Birthday Cake (blow candles)
- **Page 7**: Gift Reveal (video + gift)

## Customization

Edit `/src/data/personalInfo.js` to add your personal information.

## Tech Stack

- React 18
- Vite
- Framer Motion
- Tailwind CSS
- Howler.js (audio)
- Canvas Confetti
